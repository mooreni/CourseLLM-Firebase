# Teacher Management Spec

## Requirements

### Requirement: Course Management
Teachers SHALL be able to view and manage the courses they teach.

#### Scenario: Dashboard View
- **WHEN** a teacher visits the teacher dashboard
- **THEN** they see a list of their courses with "Manage" and "Reports" options

### Requirement: Course Editing
Teachers SHALL be able to edit course details, materials, and learning objectives.

#### Scenario: Edit Learning Objectives
- **WHEN** a teacher opens the "Learning Objectives" tab for a course
- **THEN** they can update the text for objectives, skills, and trajectories
- **AND** save the changes to the system

#### Scenario: Manage Materials
- **WHEN** a teacher opens the "Course Materials" tab
- **THEN** they can view existing files (PDF, PPT) and upload new ones
