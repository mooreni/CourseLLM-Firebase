# Glossary

- **Chunk:** A small, coherent unit of markdown content that covers a single topic.
- **Document:** A single item within the search index, corresponding to a chunk.
- **Index:** The data structure that stores the searchable content.
- **Full-Text Search:** A search technique that examines all of the words in every stored document as it tries to match search criteria.
- **REST API:** A web service that uses RESTful architecture, providing a way for different applications to communicate over HTTP.
- **`bm25s`:** A Python library for fast and efficient full-text search.
- **FAISS:** A library for efficient similarity search and clustering of dense vectors.
- **Chroma:** An open-source embedding database.
- **DSPy:** A framework for algorithmically optimizing language model prompts and weights.
