# User Stories

## As a Student

- **I want** to search for a specific topic across all my courses **so that I can** quickly find the information I need for my studies.
- **I want** the search results to show a snippet of the text around the keyword **so that I can** quickly determine the relevance of the result.
- **I want** the search to be fast **so that I can** get the information I need without waiting.

## As a Teacher

- **I want** to be able to search for specific content in my courses **so that I can** easily reuse and reference materials.
- **I want** to be able to update the search index when I update my course materials **so that** the search results are always up-to-date.
